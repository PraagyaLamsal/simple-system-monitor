# System Monitor

A real-time system monitoring program that collects and displays live CPU, GPU, memory, disk, and network usage data.  
Built with Node.js backend + React frontend.

## Features
- Live monitoring of:
  - CPU usage (overall and per core)
  - GPU utilization, memory usage, and temperature
  - System memory (RAM) usage
  - Disk usage per drive
  - Network throughput (RX/TX)
- Real-time data using WebSockets
- Lightweight React frontend with live updates
- One-click startup using a batch script

### Prerequisites
- Node.js (LTS)
